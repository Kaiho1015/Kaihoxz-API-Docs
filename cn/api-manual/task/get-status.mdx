## 概述

视频、图片生成等需要较长时间处理的操作属于**异步任务**。调用生成接口后，系统会返回一个 `task_id`（格式如 `task-wave17486161910001884721`），您需要通过本接口轮询查询任务的状态和生成结果。

<Warning>
**关于结果有效期**：生成结果的下载链接有效期通常为 **3 天**（`expires_at` 字段标示过期时间戳），过期后链接将失效，请及时将文件下载到您本地或您的服务器。
</Warning>

## 请求参数

<ParamField path="task_id" type="string" required>
  任务的唯一标识符。此 ID 在调用生成接口成功后，在返回结果的 `data` 字段中提供。

  格式示例：`task-wave17486161910001884721`
</ParamField>

## 任务状态流转

```
提交成功 → queued → processing → succeeded
                                → failed
```

| 状态 | 说明 | 是否有 result |
|------|------|:---:|
| `queued` | 任务已提交，排队等待处理 | ❌ |
| `processing` | 正在生成中 | ❌ |
| `succeeded` | 生成完成，可获取结果 | ✅ |
| `failed` | 生成失败 | ❌ |

## 轮询代码示例

<Warning>
**轮询建议**：排队阶段间隔 5-8 秒，处理阶段间隔 3-5 秒。不建议间隔低于 2 秒，否则可能触发限流。
</Warning>

```python Python theme={null}
import time
import requests

def poll_task_result(task_id, api_key, timeout_seconds=600):
    """轮询任务结果，支持超时控制"""
    url = f"https://ai.kaiho.cc/v1/tasks/{task_id}"
    headers = {"Authorization": f"Bearer {api_key}"}

    start_time = time.time()

    while True:
        if time.time() - start_time > timeout_seconds:
            raise TimeoutError("查询任务超时")

        res = requests.get(url, headers=headers).json()

        # 判断请求是否成功
        if res.get("code") != "success":
            raise Exception(f"请求失败: {res}")

        data = res["data"]
        status = data["status"]

        print(f"当前状态: {status}")

        if status == "succeeded":
            return data["result"]
        elif status == "failed":
            raise Exception(f"任务执行失败: {data}")

        # 根据状态动态调整轮询间隔
        if status == "processing":
            time.sleep(3)   # 处理中，轮询快一点
        else:
            time.sleep(6)   # 排队中，轮询慢一点

# 使用示例
# result = poll_task_result("task-wave17486161910001884721", "YOUR_API_KEY")
# print(result)
# 图片任务: {"images": [{"url": ["https://..."], "expires_at": 1749345600}]}
# 视频任务: {"videos": [{"url": ["https://..."], "expires_at": 1749345600}]}
```

```javascript JavaScript theme={null}
async function pollTaskResult(taskId, apiKey, timeoutMs = 600000) {
  const url = `https://ai.kaiho.cc/v1/tasks/${taskId}`;
  const startTime = Date.now();

  while (true) {
    if (Date.now() - startTime > timeoutMs) {
      throw new Error("查询任务超时");
    }

    const res = await fetch(url, {
      headers: { "Authorization": `Bearer ${apiKey}` }
    });
    const json = await res.json();

    if (json.code !== "success") {
      throw new Error(`请求失败: ${JSON.stringify(json)}`);
    }

    const { status, result } = json.data;
    console.log(`当前状态: ${status}`);

    if (status === "succeeded") return result;
    if (status === "failed") throw new Error(`任务失败: ${JSON.stringify(json.data)}`);

    // 动态轮询间隔
    await new Promise(r => setTimeout(r, status === "processing" ? 3000 : 6000));
  }
}

// 使用示例
// const result = await pollTaskResult("task-wave17486161910001884721", "YOUR_API_KEY");
```

## 错误响应

当请求本身出错时（如 task_id 不存在），返回格式如下：

```json theme={null}
{
  "code": "task_not_exist",
  "message": "task_not_exist",
  "data": null
}
```

常见错误码：

| code | HTTP 状态码 | 说明 |
|------|:---------:|------|
| `task_not_exist` | 400 | 任务 ID 不存在或不属于当前用户 |
| `authentication_error` | 401 | API Key 无效 |
| `get_task_failed` | 500 | 服务器内部错误 |
